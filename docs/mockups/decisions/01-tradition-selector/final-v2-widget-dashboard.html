Decision 1 -- Final v2: Customizable Widget Dashboard

The dashboard is tradition-agnostic. Users build their own view from any tradition. There is no global tradition selector on the dashboard. Instead, users pin widgets from any tradition and arrange them freely. The "+ Add Widget" button opens a categorized catalog. We provide sensible defaults (Vedic-primary with some Western and Chinese) that cover the most common use case.

The tradition/system selector has been moved to the Charts view, where it makes more contextual sense -- when you're creating or analyzing a chart, that's when you choose a tradition. The dashboard is just your personalized command center, mixing whatever you care about.

Each widget card shows a tradition badge (color-coded: Vedic=gold, Western=blue, Chinese=red, AI=purple), a drag handle for reordering, and a subtle remove button on hover. This replaces the two-mode Focus/Explore approach from the previous iteration.
