  • Planet glyphs with exact degree positioning

Option C: User-Selectable Format with Smart Defaults
User-selectable format with smart defaults. Auto-detects based on the user's language/location preference. Tamil/South Indian users see South Indian by default, North Indian users see North Indian, Western users see the wheel. Users can switch anytime via the segmented control above the chart, or change their default in Settings. Most inclusive approach. Slightly more complex to build but serves all audiences. The comparison strip below the chart helps users discover the other formats.
