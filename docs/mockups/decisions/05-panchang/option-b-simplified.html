☽
No dedicated Panchang page. The "Today's Sky" card above is the only panchang surface in this option. Panchang is grayed out in the sidebar. Users who want muhurta timelines, choghadiya tables, or festival calendars would need to use external tools like Drik Panchang.
Design Decision Analysis

Minimal panchang. Just a dashboard card -- no dedicated page. Quick daily context without building a full panchang system. Easy to implement -- just display the basic panchang data your API already calculates. But power users and daily panchang checkers will find it insufficient. No muhurta timeline, no choghadiya, no festival data.
