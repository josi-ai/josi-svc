Design Annotation — Final Decision: AI Chatbot + Card-Based Insights

Architecture: The AI chatbot is the primary experience — users converse naturally about their chart. The "Generated Insights" tab provides a library of AI-created reference content. Each insight links back to the chatbot for deeper exploration via "Chat about this" links. This makes AI the central feature, not just a content generator.

Why chatbot-first: A conversational interface lets users ask the questions that matter to them in the moment, rather than passively consuming pre-generated content. The chat becomes the single entry point for all AI-powered features: readings, dasha analysis, transit interpretation, compatibility, predictions, and remedies. Users don't need to navigate to separate pages for each concern — they simply ask.

Chart context selector: The dropdown in the tab bar lets users switch which chart the AI references. Placed alongside the tabs, it provides persistent context awareness without taking up vertical space in the chat header. This enables conversations about any saved chart — their own birth chart, a partner's chart, a child's chart, or even an event chart. The AI's responses are always grounded in real chart data, not generic horoscopes.

Generated Insights as reference library: The secondary tab aggregates all AI-generated content into scannable cards. Filters by type (Reading, Prediction, Transit, Dasha) enable quick browsing. Cards are deliberately concise — 2-3 lines max, with key metrics and visual indicators (progress bars, energy pills, compatibility scores). Each card's "Chat about this" link switches to the chatbot tab with the topic pre-loaded, creating a natural flow between reference and conversation.

Design details: AI messages use DM Serif Text for astrological content and Inter for structural headers, creating a warm, authoritative tone. User messages use a slightly different background color for visual distinction. The chat area has a subtle gradient fade at the top indicating scroll history. Suggestion chips below the welcome message reduce friction for new users who may not know what to ask. The chat input includes a "+" button for referencing specific charts within a conversation.

Engagement model: The chatbot drives deep engagement (5-15 minute sessions with back-and-forth conversation). The insight cards drive frequent engagement (quick daily check-ins). Together they create two engagement loops: depth (chatbot) and breadth (cards). Users who start with cards often transition into chat for deeper exploration. Users who start with chat build up a library of insights they can reference later.

Monetization: Free users get limited chat messages per month and basic insight generation. Paid tiers unlock unlimited chat, more detailed AI responses (longer context windows, more chart data included), and automatic insight generation (weekly transit cards, monthly dasha updates). The chatbot's natural conversational flow also creates organic upsell opportunities: "I can give you a deeper analysis of this transit — this is available on the Mystic plan."
