## A/B Testing Plan

### Phase 1: Narrow to Top 5
- Internal team vote on emotional resonance
- Filter for uniqueness (trademark search)
- Filter for cross-cultural appeal (US + India)

### Phase 2: Quantitative Testing
- Landing page split tests (hero section tagline)
- Social media ad variants (Instagram, X)
- App store listing variants (if applicable)

### Phase 3: Metrics
- Click-through rate (CTR)
- Time on page
- Sign-up conversion rate
- Brand recall (survey after 48h)

---

## Evaluation Criteria

| Criterion | Weight | Description |
|-----------|--------|-------------|
| Memorability | 25% | Can someone repeat it after hearing once? |
| Emotional resonance | 25% | Does it create a feeling? |
| Brand fit | 20% | Does it sound like the Alchemist archetype? |
| Cross-cultural | 15% | Works for Chennai and San Francisco? |
| Uniqueness | 15% | No other brand owns this phrase? |
